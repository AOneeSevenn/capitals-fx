# ZL-CAPITALS.FX - Website Komunitas Trading Forex

Website komunitas trading forex premium dengan fitur lengkap untuk edukasi, sinyal trading, dan tools profesional.

## Fitur Utama
1. **Homepage Modern** dengan live market widget
2. **Sistem Keanggotaan** dengan 3 tier (Basic, Pro, VIP)
3. **Dashboard Member** dengan tracking portfolio
4. **Forum Komunitas** untuk diskusi trader
5. **Halaman Edukasi** dengan video course
6. **Live Sinyal Trading** real-time
7. **Responsive Design** untuk semua device

## Teknologi
- HTML5, CSS3, JavaScript Vanilla
- Font Awesome Icons
- Google Fonts (Poppins & Montserrat)
- No dependencies framework (lightweight)

## Cara Install
1. Download semua file
2. Upload ke hosting/server
3. Update informasi kontak di footer
4. Ganti gambar placeholder di folder `img/`
5. Sesuaikan warna di file `style.css` jika perlu
6. Deploy dan online!

## Halaman yang Tersedia
- `index.html` - Homepage utama
- `dashboard.html` - Dashboard member
- `pricing.html` - Halaman pricing
- `forum.html` - Forum komunitas
- `signals.html` - Sinyal trading
- `education.html` - Edukasi trading
